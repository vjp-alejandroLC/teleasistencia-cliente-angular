«TIC-TAPSD. La vida conectada». El proyecto en el que se basa el cliente web de teleasistencia TLA.

TIC-TAPSD es un proyecto colaborativo entre el ciclo formativo de TAPSD del IES “San Martín” de Talayuela y el Departamento de Informática del IES “Valle del Jerte” de Plasencia.

El proyecto consiste en el diseño, creación y explotación de una plataforma software formada por una aplicación web y una aplicación móvil, que sirva a los alumnos del módulo de Teleasistencia para realizar las prácticas necesarias para la capacitación en su titulación, además de servir para los alumnos de informática como proyecto para desarrollar sus capacidades.

«Teleasistencia». Entendiendo el concepto principal de este servicio.

La teleasistencia domiciliaria es un servicio que permite a las personas mayores y/o personas discapacitadas, entre otros colectivos vulnerables, contar con ayuda de personal preparado para dar respuesta a sus necesidades durante 24 horas al día, todos los días del año. La comunicación entre los usuarios y el personal de los centros de atención a personas dependientes se realiza mediante el uso de una línea de teléfono y una infraestructura de comunicaciones e informática especifica.

Los teleoperadores del centro de atención cuentan con una serie de recursos humanos y materiales, propios del usuario o existentes en la comunidad, con los que dar solución a las necesidades del paciente.

Así mismo, este servicio se complementa con una agenda de paciente, utilizada para recordarle al mismo la realización de una tarea en un momento determinado, de forma esporádica o con cierta periodicidad.
